# SnapServe Landing Page

Ultravox.ai-inspired bento-grid landing page for SnapServe.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Design

- Left sidebar navigation (Ultravox layout)
- Bento-box grid with 1px `#27272a` borders
- Dot-pattern page background
- Glowing gradient CTA buttons
- Spectrogram, pipeline, and memory-stack visuals
- Scatter comparison chart + platform mockup

## Stack

Vite · React · Tailwind CSS v4
